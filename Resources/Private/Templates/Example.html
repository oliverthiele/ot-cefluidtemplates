Example Fluid Template based conversion

This extension is intended for recurring elements, which can be inserted by the editors on different pages and should get a centrally defined layout.
But it can also be used as a replacement for the HTML element. This is often more useful for several reasons:

  • Changes can be managed with Git.
  • Editors can no longer include arbitrary HTML code.
  • It's easier to find strings in an IDE than in the TYPO3 database.
  • The Fluid ViewHelpers can be used.
